# Wake Word Datasets

## Purpose

This directory contains training and evaluation datasets for the custom VoiceLab wake word.

## Status

No datasets collected yet. Dataset collection plan and privacy guidelines must be defined before populating this directory.

## Guidelines

- All audio samples must be collected with explicit consent from participants.
- Do not commit raw audio of identifiable individuals without a documented consent process.
- Negative examples (non-wake-word speech, background noise, TV audio) are required for FAR evaluation.
- Document the collection conditions (room type, device, noise level) for each sample batch.
